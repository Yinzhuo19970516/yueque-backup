# day27 vim visual mode
## 进入 visual mode

v 按字母  
V 按行,webstrom 视觉效果非常棒  
<ctrl-v> 按块

<ctrl-c> 退出模式

## 如何给三行内容行尾添加字符 HELLO

G 光标切到文件最后  
ctrl+v  
kkk 选三行  
$ 光标切到行尾  
A 光标之后插入，进入 insert  
hello  
esc 退出
